## Sobre o projeto :computer:

Uma aplicação web que é capaz de realizar cálculos que são medidas centrais na área de estatísticas, as quais são: 
 - Média Artimética Simples;
 - Média Aritmética Ponderada;
 - Moda;
 - Mediana.
